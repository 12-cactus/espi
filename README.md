# Espinoso

## TL;DR

```sh
git clone git@github.com:12-cactus/espi.git && cd espi
npm install
npm run dev
```

`npm run dev` command starts bot in _polling mode_.
If you want to run as _webhook mode_
you have to make a tunnel and then start.

Console one:

```sh
npm run tunnel
```

Console two:

```sh
npm start
```
